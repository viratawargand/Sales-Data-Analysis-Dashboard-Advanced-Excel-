# Sales-Data-Analysis-Dashboard-Advanced-Excel-
This project analyzes sales data containing 500+ records and 15 columns using Advanced Microsoft Excel. The objective was to transform raw sales data into an interactive dashboard that helps users monitor sales performance, identify trends, and make data-driven decisions.

## Project Features

### KPI Metrics

* Total Revenue: ₹64.12 Lakhs
* Total Orders: 500
* Total Quantity Sold: 2829
* Average Order Value (AOV): ₹128,245

### Interactive Components

* Product-wise Slicer
* Region-wise Slicer
* Sales Channel Filter
* Payment Method Filter

### Visualizations

* Sales Trend Analysis (Line Chart)
* Sales Channel Comparison (Bar Chart)
* Payment Method Distribution (3D Pie Chart)
* Product Performance Analysis
* Regional Sales Comparison
* Area Chart for Quantity Analysis

## Excel Skills Applied

* Pivot Tables
* Pivot Charts
* Slicers
* Dashboard Design
* KPI Calculations
* Data Aggregation
* Data Visualization
* Interactive Reporting

## Business Insights Generated

* Identified top-performing products and regions.
* Analyzed sales patterns across different sales channels.
* Evaluated customer payment preferences.
* Measured overall business performance using KPIs.
* Created a user-friendly dashboard for decision-making.

## Outcome

Successfully designed a professional sales dashboard that enables stakeholders to monitor key business metrics and explore sales performance through interactive filters and visual analytics.
